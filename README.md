# Chatbox

## Тестовое задание: Real-time Chat Application

Разработать веб-приложение для обмена сообщениями в реальном времени. Опционально можно добавить функционал видео-историй.

## Технологический стек

* **Backend:** NestJS + TypeScript;
* **Frontend:** SvelteKit + TypeScript;
* **Database:** MongoDB;
* **Real-time:** Socket.io/WebSocket.
